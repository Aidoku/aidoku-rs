# buny-cli

A command-line utility for Buny source development and testing.

To get started, run `cargo install --git https://github.com/BunyApp/buny-rs buny-cli`.

## Usage

```ignore
Usage: buny <COMMAND>

Commands:
  package  Build and package a source
  build    Build a source list
  init     Initialize a new source
  serve    Serve a source on the local network
  verify   Verify a source is ready to be published
  help     Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
