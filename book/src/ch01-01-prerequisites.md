## Prerequisites

Since aidoku-rs is the only maintained wrapper for the Aidoku source API, all sources are written
using the [Rust](https://rust-lang.org/) programming language. In-depth knowledge of Rust isn't
necessary, but familiarity with other programming concepts is. For a guide to learning more about
Rust, check out the [Rust book](https://doc.rust-lang.org/book/).

Familiarity with HTML and CSS selectors is also necessary when creating sources that rely on web
scraping.

Additionally, you will need a physical iOS device or iOS simulator installed through Xcode for
running sources.
